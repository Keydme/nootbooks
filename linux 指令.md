## command
df -h 查看内存
ls 看当前目录下的可见文档
ls -al 查看所有文件详细信息
kill -9 强制杀死
alias la "ls -al"
chmod 777 设置至高无上的权利
ps -ef 查看所有进程(process)
date +%H 显示当前小时
whoami 查看用户名
grep 查找指定行


neofetch 查看版本等好玩信息
